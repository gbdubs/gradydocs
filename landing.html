GradyDocs

Collaborate with Anyone, Anywhere.

What is it?

A simple collaboration tool for text documents.

Creating a Document

Enter in a phrase to describe your document below.

Sharing

Send someone else the link.

Saving

We do this for you. But we provide no guarentees that your file will be there in a few months, so only use this as a temporary collaborative tool.

Is it secure?

Nope! Anyone with the link can edit your document.

Is it stable?

Maybe? But don't count on it.

Why?

I wanted to mess around with NodeJS and refresh myself on some finer points of Javascript. The result is only a few kilobytes big, super simple to use, and doesn't have too many bugs.
